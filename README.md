# Banking AI Dashboard Design

Professional Banking Dashboard with AI Insights and Financial Analytics.

## User Account (Default)
- **Email**: admin@vbac.com
- **Password**: admin123

## Tech Stack
- **Frontend**: Vite + React + TypeScript + Recharts + Tailwind CSS
- **Backend**: Python (FastAPI) + DuckDB

## Project Structure
- `/frontend`: Frontend source code (Vite + React)
- `/backend`: Backend API services and database
- `/data`: Financial datasets

## Development Setup

### 1. Prerequisites
Install Node.js and Python 3.10+.

### 2. Frontend Setup
```powershell
npm install
npm run dev
```

### 3. Backend Setup
Navigate to the `backend` folder and follow the instructions in its README, or use the root shortcut:
```powershell
npm run backend
```

## Available Scripts
- `npm run dev`: Starts the Vite development server
- `npm run build`: Build production bundle
- `npm run backend`: Starts all backend services (PowerShell required)
- `npm run backend:stop`: Stops all backend services
